Sửa Lỗi Phân Quyền Firestore Database

Hệ thống lưu trữ dữ liệu báo lỗi phân quyền Missing or insufficient permissions. Để mở khóa và kích hoạt mây đồng bộ, em hãy thực hiện các bước cấu hình Rules đơn giản sau:

  1. Đăng nhập trang quản trị Firebase Console.
  2. Ở menu đứng bên trái màn hình, chọn mục Firestore Database.
  3. Chọn thẻ Rules (Quy tắc) ở góc trên cùng.
  4. Sao chép toàn bộ mã quy tắc ở hộp phía dưới, xóa sạch mã cũ rồi dán mã mới thay thế, sau đó nhấn Publish.
MÃ RULES PHÙ HỢP CẦN DÁN: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /artifacts/{appId}/public/data/{document=**} {
      allow read, write: if true;
    }
  }
}
